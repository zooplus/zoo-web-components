# Checkbox

Checkbox component which provides styles for `<input type="checkbox"/>`.

## Attributes
|   **Name**    |           **Description**                                                      | **Type** | **Default** | **Optional** |
| :-----------: | :-----------------------------------------------------------------------------:|:--------:|:-----------:|:-----------:|
| `labeltext`   | text to be presented on the right side of the checkbox                         | text     |             |   ✔
| `valid`       | flag which indicates whether the input is valid or not                         | boolean  |   `true`    |   ✔
| `highlighted` |flag indicating whether the outline around the input should be visible (border) | boolean  |   `false`   |   ✔
|`inputerrormsg`| text that should be displayed when `valid` is set to `falsy` value             | text     |             |   ✔
|   `infotext`  | text that should be displayed as additional info below the checkbox            | text     |             |   ✔
## CSS Variables
|        **Name**        |   **Default**  |
| :--------------------: |:--------------:|
| `--primary-mid`        |    `#3C9700`   |
| `--warning-mid`        |    `#FF6200`   |
## Slots
|      **Name**    |           **Description**          |
| :--------------: | :---------------------------------:|
| `checkboxelement`| `<input id="checkbox-id" slot="checkboxelement" type="checkbox/>`|
| `checkboxlabel`  | `<label for="checkbox-id" slot="checkboxlabel">An example checkbox</label>`|

## Example usage:
```
<zoo-checkbox highlighted="true">
	<input id="checkbox-id" slot="checkboxelement" type="checkbox"/>
	<label for="checkbox-id" slot="checkboxlabel">An example checkbox</label>
</zoo-checkbox>
```
### Rendered
<zoo-checkbox highlighted="true">
	<input id="checkbox-id" slot="checkboxelement" type="checkbox"/>
	<label htmlFor="checkbox-id" slot="checkboxlabel">An example checkbox</label>
</zoo-checkbox>
